Simple Car Sales System

This is a simple project on Car Sales programmed in JAVA. The purpose of the project is to build an application program to reduce the manual work for managing Car Models, Car Sales, Car Details, Car Search etc. This project provides searching facilities based on various factors such as Car Models, Car information etc. This inturn increases the efficiency of managing various Car Models and improved Sales.
Must Include:
  • AboutDialog
  • AddCarPanel
  • Car
  • CarDetailsComponents
  • cars data
