Using Ternary Operator

Before moving to the program, let's understand the ternary operator. It is a part of the Java conditional statement. It contains three operands. It evaluates a Boolean expression and assigned a value based on the result. We can use it in place of the if-else statement. The syntax of the ternary operator is as follows: 
variable_name = (expression) ? value if true:value if false
If the above expression returns true the value before the colon is assigned to the variable variable_name, else the value after the colon is assigned to the variable.
